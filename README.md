# spring-cloud-base

springboot、spring cloud alibaba base pom management

# 介绍

引入 Spring Boot、Spring Cloud、Spring Cloud Alibaba 三者 BOM 文件，进行依赖版本的管理，防止不兼容。 在 https://dwz.cn/mcLIfNKt 文章中，Spring Cloud
Alibaba 开发团队推荐了三者的依赖关系
